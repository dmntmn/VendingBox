# Расчёт частоты опроса сателлитов по RS-485

## Исходные данные

- Формат UART: 8N1 → 10 бит на байт
- Сателлитов: 32
- Структура пакета: `[ADDR][LEN][CMD][DATA][CRC_LO][CRC_HI]`
- Запрос: 5 байт (DATA пустой)
- Ответ: 7 байт (DATA = 2 байта максимум)
- Turnaround delay: 1 байт (переключение DE на MCU)

---

## Расчёт для 9600 baud

```
Время 1 байта: 10 / 9600 = 1.042 мс

Запрос  (5 байт): 5 × 1.042 = 5.210 мс
Turnaround       :            1.042 мс
Ответ   (7 байт): 7 × 1.042 = 7.294 мс
─────────────────────────────────────────
Обмен с 1 MCU   :            13.546 мс

Timeout (нет ответа): 5.210 + 10.000 = 15.210 мс

Цикл × 32 (все отвечают): 13.546 × 32 = 433 мс
Цикл × 32 (все молчат)  : 15.210 × 32 = 487 мс

Частота опроса: 1 / 0.433 ≈ 2.3 Гц
```

---

## Расчёт для 57600 baud

```
Время 1 байта: 10 / 57600 = 0.174 мс

Запрос  (5 байт): 5 × 0.174 = 0.868 мс
Turnaround       :            0.174 мс
Ответ   (7 байт): 7 × 0.174 = 1.215 мс
─────────────────────────────────────────
Обмен с 1 MCU   :             2.257 мс

Timeout (нет ответа): 0.868 + 5.000 = 5.868 мс
(timeout пересчитан: ответ ~1.2 мс × 4 = ~5 мс)

Цикл × 32 (все отвечают): 2.257 × 32 =  72 мс
Цикл × 32 (все молчат)  : 5.868 × 32 = 188 мс

Частота опроса: 1 / 0.072 ≈ 13.9 Гц
```

---

## Расчёт для 115200 baud

```
Время 1 байта: 10 / 115200 = 0.087 мс

Запрос  (5 байт): 5 × 0.087 = 0.434 мс
Turnaround       :            0.087 мс
Ответ   (7 байт): 7 × 0.087 = 0.608 мс
─────────────────────────────────────────
Обмен с 1 MCU   :             1.129 мс

Timeout (нет ответа): 0.434 + 2.000 = 2.434 мс
(timeout пересчитан: ответ ~0.6 мс × 3 = ~2 мс)

Цикл × 32 (все отвечают): 1.129 × 32 =  36 мс
Цикл × 32 (все молчат)  : 2.434 × 32 =  78 мс

Частота опроса: 1 / 0.036 ≈ 27.8 Гц
```

## Выводы

| Скорость | Оценка |
|----------|--------|
| 9600     | Задержка ~433 мс — заметна пользователю. Допустимо только с unsolicited BUTTON_EVENT (команда 0x07) |
| 57600    | Задержка ~72 мс — ниже порога восприятия (~100 мс). Длина шины до 200 м. Оптимальный компромисс |
| 115200   | Задержка ~36 мс — минимальная. Длина шины до 100 м. Выбор если кабель короткий |

> Порог восприятия задержки человеком: ~100 мс.
> При 57600 и 115200 нажатие кнопки обрабатывается незаметно для покупателя.
> При 9600 рекомендуется оставить unsolicited BUTTON_EVENT (0x07) как основной механизм
> оповещения о нажатии, не дожидаясь цикла опроса.
